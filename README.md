# Tema de Wordpress con Grilla Framework

**requisitos**
 - node: 14
 - npm: 6
 - php: 7
 - gulp global: [link](https://gulpjs.com/docs/en/getting-started/quick-start)

**install**
 1. clonar en la carpeta themes
 2. npm install
 3. npm run dev

**build**
 - npm run build

**plugins de wordpress**

 - GenerateBlocks
https://generateblocks.com/

 - Block Navigation
https://wordpress.org/plugins/block-navigation/
